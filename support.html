Quick Troubleshooting Guide

Follow these step-by-step instructions if you run into any common platform operational issues:

  1. 1.

    Gateway Content Mismatch / Outdated Data Cache

    1. 1.1.

      Perform a hard refresh on your web browser (Ctrl + F5 or Cmd + Shift + R).

    2. 1.2.

      If utilizing a local IPFS node companion, run an updated hash pinning command.

  2. 2.

    Unresolved DNS Lookup for .NFT Domain Extension

    1. 2.1.

      Ensure your network uses a Web3-compatible DNS service like Cloudflare (1.1.1.1) or Google DNS.

    2. 2.2.

      Alternatively, install the official Unstoppable Domains extension or access through an standard HTTP gateway link.
